# Indexed Documentation

Welcome to the Indexed documentation! This directory contains comprehensive guides for both users and developers.

## Overview

Indexed is a privacy-first, local semantic search engine that helps you make your institutional knowledge searchable and accessible to AI agents. All processing happens locally on your machine—no data leaves your infrastructure.

## Contributing to Documentation

Documentation improvements are welcome! When contributing:

1. **Maintain consistency:** Follow the existing style and tone
2. **Test examples:** Ensure all command examples work as written
3. **Update links:** Verify all internal links point to existing files
4. **Use `.mdx` format:** All documentation files use `.mdx` extension

### Documentation Style

- Use clear, user-friendly language
- Provide practical examples
- Include troubleshooting tips where relevant
- Link to related sections
- Use code blocks with syntax highlighting
